#Configuring Our RPi

This first install assumes use of Debian Jessie.

1.  Install to SD Card
2.  Boot RPi and Enable SPI.
3.  <code>Sudo apt-get update && apt-get upgrade<code> 
4.  Install RPi LAMP Stack.
